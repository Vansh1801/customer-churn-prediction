# Customer Churn Prediction & Analysis

## Overview

This project aims to predict customer churn and analyze key factors affecting customer retention using machine learning techniques.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## Project Workflow

1. Data Loading
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Encoding
5. Model Building (Logistic Regression, Random Forest)
6. Model Evaluation

## Key Insights

* Customers with lower tenure are more likely to churn
* Higher monthly charges increase churn probability
* Certain service features influence churn behavior

## Model Performance

* Logistic Regression and Random Forest models were used
* Evaluated using accuracy, confusion matrix, and classification report

## Conclusion

The project helps in identifying high-risk customers and provides insights to improve customer retention strategies.
